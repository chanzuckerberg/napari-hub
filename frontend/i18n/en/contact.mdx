# Contact

For help with or ideas for the napari hub, please [check out the discussion board]({HUB_REPO}/discussions).

For bugs or features requests, please [submit an issue on our GitHub repository]({HUB_REPO}/issues).

For issues related to napari, please visit its [image.sc forum]({IMAGESC}) or [GitHub]({NAPARI_REPO}/issues) site.

If you believe you have found a security issue with the napari hub, please responsibly disclose by contacting the CZI security team at [security@chanzuckerberg.com](mailto:security@chanzuckerberg.com).

For all other inquiries, email the hub team at [team@napari-hub.org](mailto:team@napari-hub.org).
